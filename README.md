# Configuração de proxy

Para configurar o proxy, siga as instruções

* Abra o git bash
* Digite o comando:

```
git config --global http.proxy http://user:senha@172.22.0.252:8080
```

user = seu usuario de acesso
senha = sua senha de acesso
